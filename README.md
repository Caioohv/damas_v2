# damas_v2
Just a checkers game, but in haskell :)

### Instructions:
Before all, you need to clone this repo, or download its content.

then, if using bash, just do:
```sh
./compile
```

or, if using powershell/cmd/...
```sh
ghc -o checkers.exe game.hs
./checkers.exe
```

and it will run. Have fun :)
